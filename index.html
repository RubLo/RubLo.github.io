01 // about

I'm a full-stack engineer who's spent the last few years moving from frontend into applied AI. I like owning a feature across the whole stack rather than handing it between layers.

At Tourlane I started on the customer-facing product, grew into full-stack backend work, and now build the AI systems behind trip planning — a transport-recommendation service I designed and own end to end, from the Snowflake/Dagster pipeline through the LLM reasoning and evals to the API and the React UI.

The way I work: eval-driven development for AI behavior, TDD/BDD for code — nothing ships that didn't have a failing check first. Latency, cost and reliability matter as much as accuracy. Most of all, I enjoy building AI products that make it to production and get used.
